# DBS Dashboard - High-Quality Static Maps

## Quick Start

This directory contains **19 publication-ready static maps** (300 DPI) visualizing Deep Brain Stimulation (DBS) access disparities across Canada.

### Contents
- **7 Analytical Maps** - Canada-wide perspectives
- **12 Provincial/Territorial Maps** - Regional detail
- **Complete Documentation** - Technical summaries and delivery reports
- **Visual Index** - Contact sheet showing all maps

### Key Files
- `MAP_INDEX_CONTACT_SHEET.png` - Visual overview of all 19 maps
- `FINAL_DELIVERY_REPORT.md` - Complete delivery documentation
- `MAPS_GENERATION_SUMMARY.md` - Technical specifications
- `generate_all_20_static_maps_final.py` - Generation script (reproducible)

### File Naming
- Analytical maps: `static_map[N]_[name].png`
- Provincial maps: `static_map_province_[CODE].png`
- Special maps: `static_map_[name].png`

### Specifications
- **Resolution:** 300 DPI (publication-ready)
- **Color Scheme:** Professional Aquamarine & Charcoal
- **Format:** PNG with RGBA support
- **Total Size:** 5.30 MB (19 maps)
